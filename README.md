# 38Bear 銷售頁

38 熊千層蛋捲品牌網站單頁專案，使用純 HTML/CSS/JavaScript 製作。

## 專案內容

- `index.html`：主頁面
- `images/`：網站使用素材（商品圖、形象圖、社群圖等）

## 本機預覽

直接用瀏覽器開啟 `index.html` 即可預覽。

## 線上商店連結

- 港澳商店：https://38bear.com/hkshop/
